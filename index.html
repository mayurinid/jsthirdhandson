Question 6.Print the Pattern.
Write a program which ask user for no of lines and print a pattern using an asterik .
Answer 6. 
const Print_pattern = (N) => 
{

let string = "";
for (let i = 1; i <= N; i++) {
  for (let j = 0; j < i; j++) {
    string += "*";
  }
  string += "\n";
}
console.log(string);
};
